# Ollama Gateway Integration

**Type:** architecture
**Tier:** permanent
**Confidence:** 0.93
**Created:** Mon Feb 09 2026 11:30:59 GMT-0300 (Brasilia Standard Time)

The Ollama Gateway (v8) is responsible for interacting with the Ollama models, handling requests, and managing model inference. It's configured via environment variables and integrates with the Supabase vault for secure access to model keys.
