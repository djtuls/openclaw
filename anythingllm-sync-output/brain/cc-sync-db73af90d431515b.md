# Memory Scopes: User-Private and Project-Scoped

**Type:** concept
**Tier:** permanent
**Confidence:** 0.88
**Created:** Mon Feb 09 2026 13:31:07 GMT-0300 (Brasilia Standard Time)

---

Memory entries can be categorized into user-private (accessible only to the user), project-scoped (shared within a project), or shared (accessible to all users).
