# Plan Task Workflow Rule

**Type:** concept
**Tier:** permanent
**Confidence:** 0.95
**Created:** Mon Feb 09 2026 15:00:25 GMT-0300 (Brasilia Standard Time)

---

All plan reports must be attached to a Notion task in the Tasks DB. The workflow involves creating a parent task, adding sub-items for human input, and tracking the task's lifecycle.
