# MCP Update Tool Limitations

**Type:** concept
**Tier:** permanent
**Confidence:** 0.98
**Created:** Mon Feb 09 2026 15:47:26 GMT-0300 (Brasilia Standard Time)

---

The `notion-update-page` tool does not support multi-select arrays or comma-separated strings for `update_properties`, requiring the `mcp__MCP_DOCKER__API-patch-page` tool with a specific JSON format.
